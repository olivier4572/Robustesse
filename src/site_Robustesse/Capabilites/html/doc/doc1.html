DOCUMENTATION Capabilites

Utilisez cet onglet pour créer une feuille de données d'étude de capabilité machine.

Pour rappel dans ce type d'étude, il s'agit de mesures de pièces consécutives réalisées dans un intervalle de temps très court avec idéalement le même lot de matière première, le même opérateur, le même réglage machine.

Les études de capabilité machine permettent de calculer un indice de capabilité par rapport à des spécifications ou à des tolérances client. La première chose à faire est donc de renseigner en haut à gauche les limites de spécification.

Commencez par bien définir le nombre de mesures qu'il y a dans votre étude de capabilité. Pour cela, cliquez sur la valeur à droite de "Nombre de mesures" puis renseignez ce nombre dans la fenêtre qui va s'ouvrir. Ensuite, vous avez la possibilité d'ouvrir un fichier contenant vos mesures en format csv ou de faire du copier/coller avec Excel par exemple

Pour Copier le tableau, vous pouvez simplement cliquer sur "Copier les mesures vers le presse-papier" et ensuite les coller dans Excel. Pour coller des valeurs qui auraient été modifiées dans Excel, il faut d'abord les coller dans la zone prévue à cet effet qu'on fait apparaitre en cliquant sur "Afficher la zone pour copier le presse papier" puis cliquer sur "Coller les données depuis le presse-papier"

Pour réaliser des simulations, vous pouvez également cliquer sur "Modifier les mesures" ce qui vous permet d'éditer les valeurs directement dans le tableau. Ne pas oublier de cliquer ensuite sur "Valider les modifications" pour prendre en compte les nouvelles valeurs. Enfin, le bouton "Reset" est là pour remettre à zéro le tableau et se replacer dans des conditions initiales

Vous pouvez enfin enregistrer vos données dans un fichier d'extension csv pour pouvoir le conserver et retravailler dessus ultérieurement
