DOCUMENTATION Statistiques

Echelle de sigma

Utilisez l'onglet échelle de sigma pour déterminer la proportion d'individus se situant entre 2 valeurs de sigma dans une loi normale centrée réduite, c'est à dire une loi normale de moyenne 0 et d'écart type 1.

Echelle libre

Utilisez l'onglet échelle libre pour déterminer la proportion d'individus se situant entre 2 valeurs arbitraires dans une loi normale de moyenne et d'écart type que vous renseignez dans les champs de saisies.

Utlisez les curseurs pour déplacer les bornes.

Intervalle de confiance

Utilisez cet onglet pour calculer l'intervalle de confiance de la moyenne d'un échantillon qui contiendrait nb individus.

Vous pouvez modifier les valeurs de la moyenne de l'écart type et du nombre d'individus pour vous rendre compte de l'effet que cela produit sur l'intervalle de confiance.

Veuillez noter que le niveau de confiance est fixé à 95%.
