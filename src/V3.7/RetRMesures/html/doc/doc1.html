Construction-saisie

Utilisez cet onglet pour créer une feuille de données d'étude R&R aux mesures.

Commencez par bien définir le nombre d'opérateurs puis de pièces puis de répétitions et choisissez d'appliquer un ordre aléatoire pour les mesures afin de minimiser les biais dans l'étude R&R si c'est compatible avec vos contraintes de terrain.

Ensuite, vous pouvez soit :

- saisir vos données directement dans le tableau

- soit les importer à partir du presse-papier (faire apparaître la zone adéquate avec le bouton en bas)

- soit les importer depuis un fichier csv

Vous pourrez également sauvegarder vos mesures à l'aide du bouton "sauvegarder"
