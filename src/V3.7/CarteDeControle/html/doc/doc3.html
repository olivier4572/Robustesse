DOCUMENTATION Cartes de Contrôle

Utilisez cet onglet pour créer une feuille de données de carte de contrôle Xbar/R.

Pour rappel, dans ce type d'étude, il s'agit d'évaluer la stabilité dans le temps de notre process sur une période courte ou longue en différenciant la variabilité instantanée de la variabilité sur le plus long terme. Ainsi les données sont supposées avoir été acquises de la façon suivante : on prélève à intervalles réguliers ou pas des sous-groupes de pièces ou de mesures consécutives (toujours le même nombre) et on va ainsi avoir par exemple 5 mesures consécutives prises aléatoirement dans la journée pendant 30 jours de production. Ceci nous fera 150 mesures au total. Les sous-groupes de 5 mesures sont appelés "rational subgroups" dans la littérature.

Pour créer une feuille de données, commencez par bien définir le nombre d'individus dans chaque sous-groupe et le nombre de sous-groupes qu'il va y avoir dans votre carte Xbar/R. Pour cela, cliquez sur la valeur à droite de "Nombre de sous-groupes" puis renseignez ce nombre dans la fenêtre qui va s'ouvrir et faites de même avec le "Nombre d'individus par sous-groupe". Ensuite, vous avez la possibilité d'ouvrir un fichier contenant vos mesures en format csv ou de faire du copier/coller avec Excel par exemple

Pour Copier le tableau, vous pouvez simplement cliquer sur "Copier les mesures vers le presse-papier" et ensuite les coller dans Excel. Pour coller des valeurs qui auraient été modifiées dans Excel, il faut d'abord les coller dans la zone prévue à cet effet qu'on fait apparaitre en cliquant sur "Afficher la zone pour copier le presse papier" puis cliquer sur "Coller les données depuis le presse-papier"

Pour réaliser des simulations, vous pouvez également cliquer sur "Modifier les mesures" ce qui vous permet d'éditer les valeurs directement dans le tableau. Ne pas oublier de cliquer ensuite sur "Valider les modifications" pour prendre en compte les nouvelles valeurs. Enfin, le bouton "Reset" est là pour remettre à zéro le tableau et se replacer dans des conditions initiales

Vous pouvez enfin enregistrer vos données dans un fichier d'extension csv pour pouvoir le conserver et retravailler dessus ultérieurement
