DOCUMENTATION Cartes de Contrôle

Utilisez cet onglet pour créer une feuille de données de carte individuelle et étendues mobiles.

Pour rappel, ce type de carte sert généralement à étudier l'évolution de mesures sur des pièces consécutives.

Notez que dans cet outil cartes de contrôle, on ne demande pas de renseigner des tolérances. Trop souvent on utilise les cartes de contrôle pour surveiller si le process produit ou va produire des pièces hors spécifications alors que ce n'est pas la philosophie des cartes de contrôle. Une carte de contrôle sert à identifier si le process dérive. C'est un graphique qui représente la VOP (Voice of Process), c'est-à-dire ce que mon process sait faire.

Commencez par bien définir le nombre de mesures qu'il y a dans votre analyse. Pour cela, cliquez sur la valeur à droite de "Nombre de mesures" puis renseignez ce nombre dans la fenêtre qui va s'ouvrir. Ensuite, vous avez la possibilité d'ouvrir un fichier contenant vos mesures en format csv ou de faire du copier/coller avec Excel par exemple

Pour Copier le tableau, vous pouvez simplement cliquer sur "Copier les mesures vers le presse-papier" et ensuite les coller dans Excel. Pour coller des valeurs qui auraient été modifiées dans Excel, il faut d'abord les coller dans la zone prévue à cet effet qu'on fait apparaitre en cliquant sur "Afficher la zone pour copier le presse papier" puis cliquer sur "Coller les données depuis le presse-papier"

Pour réaliser des simulations, vous pouvez également cliquer sur "Modifier les mesures" ce qui vous permet d'éditer les valeurs directement dans le tableau. Ne pas oublier de cliquer ensuite sur "Valider les modifications" pour prendre en compte les nouvelles valeurs. Enfin, le bouton "Reset" est là pour remettre à zéro le tableau et se replacer dans des conditions initiales

Vous pouvez enfin enregistrer vos données dans un fichier d'extension csv pour pouvoir le conserver et retravailler dessus ultérieurement.
